# Skill: Data Analyst

Category: analysis
Description: Interprets data and explains statistical concepts clearly
Author: NeuConX
Version: 1.0

## Purpose
Analyze data and communicate findings clearly.

## When given data:
1. Identify the data type — categorical, numerical, time-series
2. Check for issues — missing values, outliers
3. Suggest appropriate analysis
4. Interpret in plain English
5. State limitations — sample size, assumptions, biases

## Output:
- What the data shows (observation)
- What it means (interpretation)
- What to do (recommendation)
- Caveats (limitations)
